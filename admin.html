Help & Support

Simple guides to help you manage your hotel efficiently.

Quick Start Guide

How to do Check-in

1. Go to Bookings tab

2. Find the guest's booking

3. Click the Status dropdown

4. Select "Active" to mark as checked-in

How to do Check-out

1. Go to Bookings tab

2. Find the guest's booking

3. Click the Status dropdown

4. Select "Completed" to mark as checked-out

Note: Record payment before marking as completed

How to Message a Guest on WhatsApp

1. Go to Bookings tab

2. Click the WhatsApp icon next to guest name

3. A message will open in WhatsApp with guest details

How to Record Payment

1. Click "Payment" button on booking card

2. Enter the amount received

3. Select payment method (Cash/UPI/Card)

4. Click "Record Payment"

How to Change Room Prices

1. Go to Rooms tab

2. Update the price for any room type

3. Scroll down and click "Save All Room Changes"

4. New price will show on the website automatically

How to Use Shift Notes

1. Go to Shift Notes tab

2. Write any important notes for the next shift

3. Click "Post Note"

4. The next person who logs in will see your note

How to Cancel a Booking

1. Go to Bookings tab

2. Find the booking to cancel

3. Click the Status dropdown

4. Select "Cancelled"

Common Tasks

Search Bookings

Use the search box at the top to find bookings by guest name or phone number.

Filter Bookings

Use the dropdown filters to show only Active, Completed, or Cancelled bookings.

View Guest History

Click the eye icon on a booking to see all previous stays by that guest.
